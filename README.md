# Restaurant – UI/UX Design

🎨 Modern mobile app interface for order foods in Sri Lanka.

## 📱 Overview
This UI/UX design showcases:
- Home page
- Menu pages
- Food Item pages
- Profile
- Cart & Checkout with Success Messsage
